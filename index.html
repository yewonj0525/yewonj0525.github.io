Projects

Things I've built

Glioblastoma Survival Prediction from Mass Cytometry via Cellular Neighborhood Features

Adapted the Cellular Neighborhood (CN) framework to non-spatial CyTOF data by constructing k-NN graphs in protein expression space. Evaluated whether CN-augmented features improve GBM patient survival prediction over a cell-type abundance baseline using Cox proportional hazards regression with leave-one-out cross-validation.

Survival Analysis CyTOF Graph ML Bioinformatics
Domain Adaptation for Legal QA: LoRA vs. DAPT

Analyzed LoRA and DAPT for legal question answering, identifying a low-resource break-even point where domain-adaptive pretraining outperforms full fine-tuning. Provides practical guidance on choosing adaptation strategies under resource constraints.

NLP LLM Fine-tuning LoRA Legal AI
SENTRY: Context-Aware Hate Speech Detection with Set-based Learning

Developed a context-aware hate speech detection model using DistilBERT and set-based learning. Achieved a macro-F1 score of 0.847 through contrastive training, improving robustness across diverse and nuanced language patterns.

NLP DistilBERT Contrastive Learning Safety AI
Multimodal Depression Detection using Clinical Text and Audio

Built a multimodal depression detection model combining clinical text and acoustic features using Bi-LSTM and CNN architectures. Improved predictive performance through window size optimization on real-world datasets.

Multimodal ML Bi-LSTM CNN Mental Health
Respiratory Disease Classification from Breathing Sounds

Built a CNN-based respiratory disease classifier using MFCC features extracted from breathing audio. Analyzed challenges including data imbalance, overfitting, and generalization — with a focus on real-world deployment considerations.

CNN Audio ML MFCC Healthcare
